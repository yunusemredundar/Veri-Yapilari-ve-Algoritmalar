## Binary Search Tree - Proje 3

[7, 5, 1, 8, 3, 6, 0, 9, 4, 2] dizisinde oluşturulacak Binary Search Tree'nin root'u olarak 5'i seçilir. Root'tan küçük sayılar sağa büyük sayılar sola yazılır.

```
  5
 / \
1   7
```

```
     5
   /   \
  1      7
 / \    / \
0   3  6   9
```

```
      5
   /     \
  1        7
 / \      / \
0   3    6   9
   / \      /
  2   4    8

```
